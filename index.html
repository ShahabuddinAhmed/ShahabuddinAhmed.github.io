House Renting

House Renting is an online platform. House owner can give house ads and see ads whose want to rent the house and contact with the house owner.

House Owner Sides

  • After creating account house owner can able to house ads.
  • House owner can upload more than one picture of the house.
  • Can be given multiple house ads.
  • After creating house ads, the owner can be updated and delete ads.
  • Also contains house cover image options.

House Renter Sides

  • Without creating an account can be able to see house ads.
  • Customize choose search options.
  • Search can be given division, location, area, taka, and room number.

Admin Sides

  • Admin can be added to a new division.
  • Admin can be added new location under a division.
  • One admin can be added to another new admin.
  • Admin can be updated his information.
  • One admin can be deleted another admin.
  • Super admin can't be deleted.

Personal Blog

A simple personal blog with separate API and UI part. The main purpose of this project is to create to write a personal post. The user can read only published post and comment on this post.

Client Sides

  • Without creating account user can read published post.
  • After creating an account user can comment on the post.

Admin Sides

  • Though it is a personal blog, Here are multiple admins exit.
  • After login user can create posts. There is a javascript based rich text editor for this purpose.
  • Each published posts are listed in chronological order in the blog.
  • Admin can be the ability to delete any comment given to his blog post.
  • Admin can be the ability to replay any comment given to his blog post.


Online Address Book

A simple online address book to use the system users can easily store their personal contact info online based system.

Main featchure

  • To use the system users have to register using his/her user name, email and password.
  • After logging in user can add, edit or delete address book entry.
  • Each address entry in address book contain commonly used fields (full name, nickname, multiple phone no., address, website, date of birth).
  • The Application has the option to download all contact information in pdf format.
